Global Constants:

  • %UserName% - Service User Name

  • %Password% - Service Password

  • %RouterPassword% - Router Password

  • %RouterIP% - Router IP Address

  • %RouterPort% - Router Port

  • %Time% - Current Time

  • %Date% - Current Date

  • %AppPath% - Application Path

  • %IP% - External IP

    Global Constants Can Be Used In Service Fields And Addresses,   

    Router Addresses, And File Paths. For  A Dynamic Logfile You

    Can Use Something Lik This: %AppPath%\Logs\%Date%.log

 

Default IP:

  • For Dial-Up Users With A Dynamic IP You Can Use Wildcards

  • ? - One Character

  • * - Anything

  • 192.*.*.* (192.32.53.32) or 192.1??.*.* (192.158.84.43)

    The Default IP Is Used By Default If LANConnect Is Not

    Checked.
